AgroVault OS

Live at: https://agrovault-nu.vercel.app


Built: A full production web application from scratch.
Deployed: Backend API on Railway (live 24/7).
Deployed: Frontend on Vercel (global CDN).

Works on any device, any browser, anywhere in the world.
No IT knowledge needed to use it.

The system is a full agricultural trade operations platform covering:
- import/export,
- QC/sorting,
- customs clearance,
- cold storage,
- financial transactions, 
- multi-level reporting (daily/weekly/monthly/quarterly/annual) for CEO, Director, Manager, Accountant, and Operator roles.

Tech stack: Python/FastAPI backend on Railway + React frontend on Vercel — both free, both running 24/7.

For the user's guide: 
- open the link,
- login as CEO,
- click through a shipment A→Z,
- view Reports, 
- view Analytics.
